Controls:
 · C - Clear Waves
 · R - Reset Simulation
 · A - Show Exposure Screen
 · H = Show/Hide HUD
 · + - Increase Brightness
 · - - Decrease Brightness
 · 0-1 - Tools

 Tools:
 0 - Clear Tool
 1 - Wave Tool
 2 - Wall Tool
 3 - Water
 4 - Glass
 5 - Diamond
